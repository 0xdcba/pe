Установка зависимостей

Установка зависимостей под linux с помощью yum может выглядеть следующим образом:

$ yum install ruby calibre rubygems ruby-devel rubygem-ruby-debug
$ gem install rdiscount

А с помощью apt-get установка зависимостей может быть произведена вот так:

$ apt-get install ruby calibre rubygems ruby-dev rake pandoc texlive-xetex libmaruku-ruby 
$ gem install rdiscount

Под Mac OS X:

$ install calibre app
$ check ruby and rubygems
$ easy_install pygments
$ port install pandoc texlive +full texlive-xetex
$ gem install rdiscount

Сборка книги из исходников

Собрать Русский перевод книги для amazon kindle:

 $ ./makeebooks ru

или

 $ FORMAT=mobi ./makeebooks ru

Собрать Русский перевод книги в формате ePub:

 $ FORMAT=epub ./makeebooks ru

Собрать Русский перевод книги в формате PDF:

$  ./makepdfs ru
